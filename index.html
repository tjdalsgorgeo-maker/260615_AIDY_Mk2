설문 설계 Agent 에디

주요 키워드를 바탕으로 설문 초안을 생성하고, 사용자 피드백을 반영해 최종 설문지와 설계 리포트를 제공합니다.

1
설계 입력 주제, 목적, 대상자 입력
2
초안 검토 문항 선택·수정·피드백
3
최종 리포트 5개 탭 산출물 생성

① 설문 설계 입력

필수 3가지만 입력해도 에디가 설문 유형과 적정 문항 수를 자동 판단합니다.

고급 옵션 열기

② 초안 검토 / 자동진단 / 피드백

에디가 만든 초안 문항을 선택·수정하고, 원하는 방향을 피드백으로 입력해 재생성할 수 있습니다.

에디가 본 설계 중점

초안 보완 포인트

초안의 강점

추천 수정 방향

초안 문항

문항을 크게 보이도록 구성했습니다. 노란색 셀은 직접 수정할 수 있고, 체크된 문항만 다음 단계로 넘어갑니다.
0개 문항
수정한 초안 문항이 반영되었습니다.

③ 최종 산출물

사용자가 확정한 최종 문항을 기준으로 척도·기법 매핑, 설계 진단, 결과 해석 가이드, 개선 로그를 제공합니다.
